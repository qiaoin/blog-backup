Typora

之前一直使用的 markdown 编辑器是 MacDown，现在却是在用着 Typora 写这篇文章（已将 md 文件默认打开方式改为 Typora 了），很喜欢这个的风格，用起来也很实在， 最喜欢的就是为了新起一个段落不用空两行（因为在预览模式下编辑，编辑器在按下一个回车键时候就已经空两行了，只是你只需要按一次回车就 ok ），因为手动回车两次空两行很是不习惯， 而使用 Typora 很符合我们编辑文档的习惯。还有很多好用的功能，例如使用 command + / 在预览和源码之间切换，当然预览和源码这两个模式都是可以编辑的。

(2017-02-18 Sat，补充) 有开启打脸模式了。Typora 确实很是出色，使用起来也很方便，但我确总是遇上莫名其妙的问题，最严重和最不能忍受的就是，在保存了一个 md 文件之后， 再次打开时它会修改我文件的格式（当然这与 Markdown 这个标记语言有关系， Markdown flavored 众多，支持的语法特性也就有着不同之处，怎样处理便是一个大问题，因此，现在 我主要使用 Emacs 的 Org-mode 来写博客和文档，复杂的文件就使用 LaTeX 来编写，Org-mode 有严格的规范，并且在 Emacs 上使用如行云流水），这就弃用。

拥有一个自己的Terminal

iTerm2 + zsh + on-my-zsh

  • iTerm2：很多人都推荐过这个终端了，也不是说MAC上自带的不好用，而是说这个东西太好用了。
  • zsh + on-my-zsh：为什么将这两个东西放在一起呢，你可以这样认为，zsh 是一个比较厉害的东西，但配置起来比较困难，可以用成语 `阳春白雪` 来做比喻，而为了其易用性，国外就有一程序员开发了一个迅速上手的 zsh 项目，称为 on-my-zsh 。

下面我们来说说如何配置已达到如下图示的效果（这里是 盗的图 ）：

1.iTerm2 的安装：直接去 这里 下载 dmg 文件就行。

2.zsh + on-my-zsh 的安装：

  • MAC上是预装了 zsh 的，这个可以通过 cat /etc/shells 查看，显示如下：
# List of acceptable shells for chpass(1).
# Ftpd will not allow users to connect who are not using
# one of these shells.

/bin/bash
/bin/csh
/bin/ksh
/bin/sh
/bin/tcsh
/bin/zsh  # <- 这个就是我们将要使用的shell
  • 设置当前用户使用 zsh： chsh -s /bin/zsh ，这样设置之后可以查看当前使用的 shell 是哪一个 echo $SHELL ，应该是 /bin/zsh
  • 安装 on-my-zsh
git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

完成之后，关闭 iTerm2，重新打开一个终端窗口就应该能够得到官网上的效果了（我们的预期）。但是这样好像离我们想要的效果还有点远，请继续看下面。

在 oh-my-zsh的github首页 上 Selecting a Theme 就是对应的主题的选择，我们发现上面的这张效果图为其默认主题 robbyrussell ，而最终效果图是 agnoster 。

4.修改 ~/.zshrc 文件，以下为前几行配置 

# Path to your oh-my-zsh installation
export ZSH=$HOME/.oh-my-zsh
# Set name of the theme to load
ZSH_THEME="agnoster"  # <- 这里就是设置主题theme
DEFAULT_USER=`whoami` # <- 特别注意这里，这里不是单引号，而是`(requires backticks, not single-quote, 就是键盘上 esc 下面的那个按键)，
                      # <- 具体看查看 https://github.com/robbyrussell/oh-my-zsh/issues/2033

这样设置之后，还会有几个小问题，还需要以下几步设置。

5.安装 Powerline字体 ，得到字体文件之后直接双击选择安装就可以了。

6.选择 Solarized配色方案 ，将这个项目 clone 下来 

git clone git://github.com/altercation/solarized.git

就会有一个 solarized 的文件夹，选择我们需要的 iTrem2 的主题，这个文件下会有两个文件，如下：

7.iTerm2 设置

  • 打开 iTerm2 -> Preferences -> Profiles -> 点选你要修改的Profile -> Colors -> 选择最下面的 Load Presets，选择 import 添加主题配置文件

browse 至以下所示的文件夹 ~/Downloads/solarized/iterm2-colors-solarized 选择 Solarized Dark.itermcolors 和 Solarized Light.itermcolors 导入， 然后选择 Solarized Dark 主题即可。

  • 切换至 Text，把 Font 和 Non-ASCII Font 都改成刚才安装好的 powerline 字体

按照网上的说法，需要将 Text Rendering 里的 Draw bold text in bright colors 的对勾给去掉。

这就是能达到上面的最终效果了，关于配置参考 Mac下配置iTerm2+oh-my-zsh 。

常用软件推荐

  • Dash：an API Documentation Browser and Code Snippet Manager。这个应用应该是我在知乎上看到的，确实不错，在这里作为第一个推荐给大家。
  • Chrome：很好用的浏览器，我现在已经很少使用 Safari 了；能够同步帐号的书签也是超实用的。
  • VirtualBox：在 MAC 上使用了破解版的 VMware 和 Parallels 之后，我最终转向了免费开源的 VirtualBox，然后就开始捣鼓安装了 ArchLinux 和 LinuxMint ，前者是能够高度定制的 Linux 系统，后者是在基于现在比较流行的 Ubuntu，并且相较 Ubuntu，我更喜欢使用 LinuxMint。
  • XMind：这个就是神器了，也就是大名鼎鼎的 思维导图 ，有什么用处，可以移步去看看知乎上的这个问题 怎么使用思维导图？ ，我一般用它来整理思路，还仿照着网上的大神整理了一份 编程入门指南 v1.4 的思维导图
  • Evernote：很是强大，基本用来同步笔记，在电脑上记录笔记，然后用手机接着看。
  • MPlayerX & VLC：这就是我在 MAC 上用到的播放器了，MPlayerX 基本就可以满足你 99% 的需求了，但有些时候你要看一些学习的视频，为了便于传播，这些视频可能是压缩了的，这时候 VLC 就可以派上用场了。(2017-02-18 Sat 补充，现在主要使用 VLC 了，因为稳定)
  • QQ & WeChat & 网易云音乐：这就是休闲常用 APP 了，网易云音乐，用它听过音乐的都说好，我会骗你么？
  • Dr.Cleaner & The Unarchiver & uTorrent & smcFanControl：这几个我就不介绍了，用来清理的、解压的、下载的，还有就是来降温的（但如果玩游戏的话，这个就没有什么用处了）。
